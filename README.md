# thmscripts
Scripts written in python for tryhackme


subdomains-enum.py
This script is used to enumerate all possible sub domains a website/domain might have.
It reads a text file that contains subdomains in list format per line and the code is ran by: python3 subdomain-enum.py vickkumar.com
This will then enumerate all subdomains on the text file with vickkumar.com to see which one exists then return the ones found.
